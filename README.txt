IMPORTANTE!!!
librerias necesarias:
pandas, imagetk, tkinter, numpy, random, image, openpyxl

de modificar el archivo EXCEL, conservar el MISMO NOMBRE: "Base2_Gabo.xlsx"

# para windows (las que no estan ya estan instaladas por default, en teoria)
pip install pandas (version 1.2 o superior)
pip install openpyxl (engine para leer xlsx con pandas)
pip install numpy

# para linux
sudo apt-install python3-pip (instalar pip)
sudo apt install python3-tk (instalar tkinter)
sudo apt-get install -y python3-pil.imagetk (instalar imagetk)
sudo pip install pandas (instalar pandas)
sudo pip install openpyxl (instalar engine)
+ comentar la linea 39 (icono de ventana) porque no se usa en linux
